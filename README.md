# 🎓 Virtual Classroom

A fully responsive, web-based virtual classroom platform built with **HTML, CSS, and JavaScript** — no frameworks, no build tools, just open and run!


## 🗂️ Project Structure

```
virtual-classroom/
│
├── index.html          # Main HTML page
├── style.css       # All styles (variables, layout, components)
├── app.js          # Course data, classroom logic, modals, toast
└── README.md
```

---

## 🚀 Getting Started

### Option 1 – Open directly
Just double-click `index.html` in your browser. No server required!

### Option 2 – Live Server (VS Code)
1. Install the **Live Server** extension in VS Code
2. Right-click `index.html` → **Open with Live Server**

### Option 3 – Clone from GitHub
```bash
git clone https://github.com/your-username/virtual-classroom.git
cd virtual-classroom
# Open index.html in browser
```

## 🛠️ Tech Stack

| Tech | Usage |
|------|-------|
| HTML5 | Page structure & semantics |
| CSS3 | Styling, variables, animations, grid/flexbox |
| JavaScript (Vanilla) | Interactivity, DOM manipulation, course filtering |
| Font Awesome 6 | Icons |
| Google Fonts | Typography (Sora + Inter) |

## 📸 Sections Overview

| Section | Description |
|---------|-------------|
| Navbar | Fixed nav with smooth scroll & active link highlighting |
| Hero | Animated classroom preview card + stats |
| Features | 6-card grid highlighting platform capabilities |
| Courses | Filterable course cards with enroll button |
| Classroom | Join/create live sessions with controls |
| About | Platform story + feature list |
| Footer | Links + social icons |

---

## 🔧 Customization

- **Add courses**: Edit the `courses` array in `js/app.js`
- **Change colors**: Update CSS variables in `:root` inside `css/style.css`
- **Add pages**: Link new HTML files from the navbar
